## 🎨 内容创作工具（11个）

### 1. andy-mp-writing ⭐⭐⭐
**功能：** Andy风格公众号文章创作系统 v2.0

**核心特点：**
- 对抗式生成（3个子Agent协作）
- Andy风格DNA（产品思维+技术类比+第一人称）
- 核心原则：讲自己+深刻利他+少营销

**适用场景：**
- AI科技解读文章
- 个人成长类内容
- 产品思维分析
- MBTI深度文章

**输入：**
- 主题/观点
- 目标读者
- 个人故事/经历
- 字数要求（默认1500-2500字）

**输出：**
- 完整的Andy风格公众号文章
- 文章质量评估
- 优化建议

**触发词：** "写公众号"、"公众号文章"、"Andy风格文章"

**优先级：** 高 - IP增长的核心工具

## ✅ 内容审核工具（3个）

### 12. content-compliance ⭐⭐⭐
**功能：** 多平台内容合规检查系统

**核心特点：**
- 支持小红书、抖音、微信公众号三大平台
- 多维度风险检测
- 风险评级系统
- 具体修改建议

**检查维度：**
1. 政治敏感
2. 色情低俗
3. 暴力血腥
4. 违法犯罪
5. 虚假信息
6. 侵权内容
7. 平台特殊规则

**风险等级：**
- 高危（必须修改）
- 高风险（强烈建议修改）
- 中风险（建议修改）
- 低风险（可选修改）
- 安全（可以发布）

**输入：**
- 待检查内容
- 目标平台（xiaohongshu/douyin/wechat/all）

**输出：**
- 风险评级
- 问题内容定位
- 具体修改建议
- 平台适配建议

**触发词：** "合规检查"、"会不会违规"、"审核内容"

**优先级：** 高 - 避免封号的必备工具

## 🗺️ 工具使用路径推荐

### 路径1：从0到1创作公众号文章
```
story-import（收集素材） 
→ story-long-scan（分析选题） 
→ andy-mp-writing（创作文章） 
→ content-compliance（合规检查） 
→ content-like-prediction（效果预测） 
→ 发布到公众号 
→ andy-zsxq-content-ops（同步到知识星球） 
→ story-long-analyze（数据复盘）
```

### 路径2：专业白皮书创作
```
story-import（收集资料） 
→ andy-product-skill（战略分析） 
→ andy-whitepaper（创作白皮书） 
→ story-review（质量审查） 
→ md2book（生成精美PDF） 
→ 发布和推广
```

### 路径3：短内容快速优化
```
story-short-analyze（分析现有内容） 
→ copywriting（优化文案） 
→ content-compliance（合规检查） 
→ 发布到小红书/朋友圈
```

### 路径4：数据驱动内容迭代
```
story-long-analyze（分析历史数据） 
→ 识别高ROI内容类型 
→ andy-product-skill（评估用户价值） 
→ story-long-scan（找类似选题） 
→ 复制成功模式创作新内容
```
