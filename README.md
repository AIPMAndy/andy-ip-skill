## 💡 核心价值

Andy IP Skill Stack 是一个**完整的个人IP增长AI工作流生态系统**：

### 🧬 基础设施层
- **DNA Memory** - 跨会话长期记忆系统，让AI真正"记住"你的IP风格
- **AndySoul** - 认知公理 + Musk工程法 + Karpathy原则的核心操作指令
- **andy-memory-loop** - 记录创作洞察，持续优化IP策略

### ✍️ 内容创作层
- **15个创作工具**：公众号、白皮书、PPT、视频、思维导图全覆盖
- **爆款方法论**：对抗式生成 + Andy风格DNA + 多平台合规检查
- **自动化引擎**：Copyflow一键生成10种风格内容

### 📱 内容发布层
- **飞书协作**：文档备份、表格管理、会议纪要等个人IP工作流工具
- **多平台适配**：公众号、小红书、抖音内容自动化发布

### 📊 数据研究层
- **情报系统**：追踪海外Builder动态，AI领域洞察
- **资源聚合**：1000+ 免费API、120+ 设计资源、黑客松信息

### 💻 开发工具层
- **全栈开发**：前端、3D、网页克隆、GitHub Pages部署
- **浏览器控制**：Kimi WebBridge实现自动化操作

## 🏗️ 架构设计

### 三层智能架构

```
┌─────────────────────────────────────────────────────────┐
│           andy-ip-skill (智能路由层)                     │
│                                                          │
│  ▸ 新手教程模式 - 交互式引导，完成第一次实际使用          │
│  ▸ 任务前路由 - 根据需求智能匹配最佳工具                 │
│  ▸ 任务后导航 - 基于结果动态推荐下一步                   │
│                                                          │
│  触发方式：/andy-ip-skill 或 /andy-ip                    │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│              5大核心领域 (能力层)                         │
│                                                          │
│  🧬 基础设施  |  ✍️ 内容创作  |  📱 平台发布             │
│  📊 数据研究  |  💻 开发工具                             │
│                                                          │
│  每个领域包含多个专业工具，支持独立调用或组合使用          │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│          40+ 子项目 (执行层)                              │
│                                                          │
│  GitHub开源仓库 + 本地Claude Skills + 飞书工具集          │
│  所有工具开箱即用，支持自定义配置和二次开发                │
└─────────────────────────────────────────────────────────┘
```

### 设计原则

1. **单步决策，动态导航** - 不预设长流程，每步根据实际结果决定下一步
2. **Skills即产品** - 每个工具独立交付价值，可灵活组合
3. **智能路由优先** - 降低工具选择成本，让AI帮你找到最佳路径
4. **实战验证优先** - 所有工具经过Andy本人验证，确保真实产生价值

## 💼 典型工作流

### 场景1：创作公众号爆款文章（完整链路）

```bash
# Step 1: 召回相关记忆和经验
/andy-memory-loop recall "公众号创作"

# Step 2: 收集灵感素材
/story-import [粘贴素材内容]

# Step 3: 分析选题方向
/story-long-scan "AI+产品"领域

# Step 4: 创作文章（对抗式生成）
/andy-mp-writing "选定的主题"

# Step 5: 合规检查
/content-compliance wechat

# Step 6: 效果预测
/content-like-prediction

# Step 7: 发布到多平台
发布到公众号 + /lark-doc（备份到飞书）

# Step 8: 数据复盘和记忆沉淀
/story-long-analyze
/andy-memory-loop remember "本次创作的关键洞察"
```

**涉及工具**: andy-memory-loop, story-import, story-long-scan, andy-mp-writing, content-compliance, lark-doc

---

### 场景2：多平台内容分发（自动化）

```bash
# Step 1: 使用Copyflow批量生成
/copyflow "一个选题" → 生成10种风格内容

# Step 2: 针对不同平台优化
小红书版本: /copywriting [AIDA框架]
抖音版本: /ai-video-producer [生成脚本]
公众号版本: /wechat-explosive-article [爆款方法论]

# Step 3: 批量合规检查
/content-compliance all

# Step 4: 生成配图
/AndyImage "内容主题" → 生成提示词 → Midjourney

# Step 5: 发布和归档
发布到各平台 + /lark-base（内容库管理）

# Step 6: 效果追踪
/lark-sheets "内容效果追踪表"
```

**涉及工具**: copyflow, copywriting, ai-video-producer, wechat-explosive-article, content-compliance, AndyImage, lark-base, lark-sheets
