# Andy IP增长工具箱 (andy-ip-skill)

> 从0到1打造个人IP的完整工具链 - 59个实战验证的Skills，覆盖IP增长8大阶段

## 🎯 这个工具箱能做什么？

### 三种使用模式

**模式A：任务前路由** - 告诉我你要做什么，我帮你找到最合适的工具
- "我想写一篇公众号文章" → 路由到 `andy-mp-writing`
- "我要发知识星球内容" → 路由到 `andy-zsxq-content-ops`
- "需要做产品决策分析" → 路由到 `andy-product-skill`

**模式B：任务后导航** - 完成当前任务后，推荐下一步最有价值的行动
- 写完文章 → 建议：合规检查 → 多平台发布 → 数据追踪
- 发布内容 → 建议：用户反馈分析 → 热点内容优化 → 商业化评估

**模式C：新手教程** - 完整的IP增长方法论和工具使用指南
- IP增长8阶段完整路径
- 每个阶段的核心工具和使用方法
- 实战案例和避坑指南

## 模式C：新手教程 - IP增长完整链路

### IP增长8大阶段

```
数据输入 → 选题策划 → 内容创作 → 多平台分发 → 用户运营 → 数据分析 → 内容优化 → 商业变现
   ↓          ↓          ↓           ↓            ↓          ↓          ↓          ↓
 灵感捕获   热点追踪   专业写作    自动发布      社群管理   效果追踪   迭代优化   变现转化
```

### 阶段1：数据输入 - 灵感捕获与素材管理

**核心价值：** 将碎片化灵感转化为可复用的内容资产

**核心工具：**
- `story-import` - 从任意来源导入灵感素材
- `andy-memory-loop` - 长期记忆管理，沉淀关键洞察

**使用场景：**
- 看到好的文章/视频 → 一键导入并提取关键观点
- 日常灵感闪现 → 快速记录并自动分类
- 用户反馈收集 → 结构化存储待分析

**操作步骤：**
1. 使用 `/story-import` 导入任意格式素材
2. 系统自动提取核心观点、标签、关联话题
3. 存入个人知识库，支持全文检索

---

### 阶段2：选题策划 - 数据驱动的选题决策

**核心价值：** 基于数据和趋势，选择高ROI的创作方向

**核心工具：**
- `story-long-scan` - 扫描和分析长文选题方向
- `andy-product-skill` - 用产品思维评估选题价值

**使用场景：**
- 每周选题会 → 分析热点趋势，评估选题潜力
- 内容规划 → 制定季度/月度内容日历
- 竞品分析 → 研究同赛道爆款内容规律

**选题评估框架：**
1. **用户价值** - 解决什么真实痛点？
2. **传播潜力** - 话题热度、争议性、共鸣度
3. **商业价值** - 能否引导转化/建立专业度
4. **创作成本** - 需要多少时间和资源

---

### 阶段3：内容创作 - 高效产出优质内容

**核心价值：** 用AI+方法论，将创作效率提升10倍以上

**核心工具（按场景分类）：**

**A. 长文创作（公众号、博客、白皮书）**
- `andy-mp-writing` - Andy风格公众号文章（对抗式生成）
- `andy-whitepaper` - 专业白皮书创作+PDF排版
- `story-long-write` - 通用长文创作工具

**B. 短文/碎片化内容**
- `story-short-analyze` - 短文分析和优化
- `copywriting` - 营销文案创作（AIDA、PAS框架）

**C. 特定类型内容**
- `andy-mbti-skill` - MBTI深度分析文章
- `handdraw-good-deed-story` - 手绘风格故事内容

**创作流程（以公众号文章为例）：**
1. 使用 `/andy-mp-writing` 启动对抗式生成
2. 系统用3个子Agent分别生成：核心观点、故事案例、金句Hook
3. 自动整合成Andy风格文章（第一人称+产品思维+技术类比）
4. 质量检查：深刻利他度、价值密度、可执行性

---

### 阶段4：内容审核 - 合规与优化

**核心价值：** 避免平台违规风险，提升内容传播效果

**核心工具：**
- `content-compliance` - 多平台合规检查（小红书/抖音/公众号）
- `content-like-prediction` - 预测内容传播效果
- `story-review` - 内容质量全面审查

**合规检查维度：**
- 政治敏感、色情低俗、暴力血腥、违法犯罪
- 虚假信息、侵权内容、平台特殊规则
- 风险评级：高危/高风险/中风险/低风险/安全

**操作步骤：**
1. 使用 `/content-compliance 小红书` 检查内容
2. 系统标注风险点并提供修改建议
3. 使用 `/content-like-prediction` 预测传播效果
4. 根据反馈优化标题、开头、金句

---

### 阶段5：多平台分发 - 自动化发布

**核心价值：** 一次创作，多平台同步发布，扩大影响力

**核心工具：**
- `andy-zsxq-content-ops` - 知识星球内容运营
- `lark-*` 系列 - 飞书文档/表格/任务管理（第三方工具，需单独集成）

**发布策略：**
- **核心平台**（公众号/知识星球）：完整版内容
- **流量平台**（小红书/抖音）：精简版+引流钩子
- **专业平台**（GitHub/个人博客）：技术深度版

**注意事项：**
- 不同平台调整标题和开头（适配算法偏好）
- 添加平台特定话题标签
- 设置引流路径（引导到核心平台/社群）

---

### 阶段6：用户运营 - 社群与互动管理

**核心价值：** 将内容消费者转化为忠实粉丝和付费用户

**核心工具：**
- `andy-zsxq-content-ops` - 知识星球内容运营和用户互动
- `andy-memory-loop` - 记录关键用户反馈和需求

**运营动作：**
- 精选评论回复 → 展示专业度和温度
- 用户问题收集 → 转化为下次选题
- 社群话题引导 → 提升活跃度
- KOC用户培养 → 扩大传播杠杆

---

### 阶段7：数据分析 - 效果追踪与优化

**核心价值：** 用数据指导创作决策，持续提升ROI

**核心工具：**
- `story-long-analyze` - 长文内容数据分析
- `andy-product-skill` - 用产品思维分析用户价值

**关键指标：**
- **内容质量**：完读率、互动率、收藏/转发比
- **传播效果**：阅读量、涨粉数、外部引用
- **商业价值**：转化率、付费率、客单价

**优化动作：**
1. 识别高ROI内容类型 → 加大投入
2. 分析低效内容原因 → 调整策略或停止
3. A/B测试标题/开头 → 提升点击率
4. 用户旅程分析 → 优化转化路径

---

### 阶段8：商业变现 - 从影响力到收入

**核心价值：** 将IP影响力转化为可持续收入

**核心工具：**
- `andy-product-skill` - 产品设计与定价策略
- `copywriting` - 销售页/落地页文案

**变现路径：**
1. **付费社群**（知识星球）- 稳定的订阅收入
2. **商业合作**（品牌投放/广告）- 高客单价但不稳定
3. **付费产品**（课程/工具/咨询）- 规模化收入
4. **RaaS服务**（Research as a Service）- 高价值企业服务

**定价策略：**
- 社群：¥199-699/年（根据提供价值定价）
- 单篇商单：¥1000-5000（根据粉丝量和影响力）
- 系统课程：¥999-4999（根据交付成果定价）
- RaaS服务：¥5000-50000/项目（根据研究深度定价）

## 模式B：任务后导航 - 推荐下一步行动

### 导航地图（完成X后，建议做Y）

| 刚完成的任务 | 推荐下一步（优先级排序） | 说明 |
|------------|----------------------|------|
| **创作完成后** |||
| 写完长文 | 1. `content-compliance` 合规检查<br>2. `content-like-prediction` 效果预测<br>3. 多平台发布 | 先确保合规，再优化传播效果 |
| 写完白皮书 | 1. `md2book` 生成精美PDF<br>2. 设计发布策略<br>3. 准备推广素材 | 专业排版提升可信度 |
| **发布完成后** |||
| 发布到公众号 | 1. 同步到知识星球（`andy-zsxq-content-ops`）<br>2. 提取金句发小红书<br>3. 监控数据表现 | 最大化内容复用价值 |
| 发布到知识星球 | 1. 引导用户互动（提问/讨论）<br>2. 精选评论收集反馈<br>3. 记录高质量问题作为下次选题 | 社群运营的核心是互动 |
| **数据分析后** |||
| 完成内容数据分析 | 1. 识别高ROI内容类型 → 复制成功模式<br>2. 优化低效内容 → 调整策略<br>3. 更新选题方向 | 数据驱动迭代 |
| **审核完成后** |||
| 合规检查通过 | 1. 立即发布到目标平台<br>2. 预设互动话题<br>3. 准备用户问题回复模板 | 趁热发布，抓住时效性 |
| 发现合规风险 | 1. 修改风险内容<br>2. 重新审核<br>3. 考虑调整发布平台 | 某些内容可能只适合部分平台 |

### 导航推荐原则

1. **价值最大化** - 优先推荐能产生最大增量价值的动作
2. **风险最小化** - 先做风险控制（合规检查），再做增长动作
3. **效率优先** - 推荐自动化/批量化的工具，避免重复劳动
4. **闭环思维** - 每个动作都应指向可衡量的结果

## 💡 核心设计理念

### 1. 单步决策，动态导航
- 不预设长流程链，每一步根据实际结果决定下一步
- 避免"一个工具走到底"的僵化流程

### 2. 工具即产品
- 每个Skill都是独立产品，有明确的输入/输出/价值主张
- 工具之间松耦合，可灵活组合

### 3. 实战验证优先
- 所有工具都经过Andy本人7个月实战验证
- 只保留真正产生价值的工具，不追求数量

### 4. 效率与质量并重
- 用AI提升效率（从8小时→30分钟）
- 用方法论保证质量（对抗式生成、多维审核）
